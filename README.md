# 🎙️ AI Voice TTS Generator

Microsoft Edge 브라우저의 무료 TTS 엔진을 활용한 **무제한 텍스트 음성 변환 웹 애플리케이션**입니다.  
Python과 Streamlit을 사용하여 제작되었으며, 별도의 API 키 없이 고품질의 AI 목소리를 생성하고 다운로드할 수 있습니다.

### ✨ 주요 기능

- **완전 무료 & 무제한:** 글자 수 제한 없이 사용 가능
- **고품질 한국어 보이스:** 선희, 인준, 현수 등 자연스러운 AI 목소리 제공
- **옵션 조절:** 말하기 속도(Rate) 조절 가능
- **즉시 다운로드:** 생성된 음성을 바로 듣고 MP3로 저장

### 🛠️ 설치 방법 (Installation)

이 프로젝트는 Python 3.9 이상 환경에서 권장됩니다.

https://github.com/kss2002/edge-TTS/releases/tag/1.1v

위 링크로 들어가서 edge.zip를 다운로드 받고, 압축을 해제합니다. <br/>
압축 해제된 폴더를 vs code나 커서 등의 에디터로 오픈합니다.

### 실행 방법

```bash
source venv/bin/activate
cd venv
streamlit run app.py
```

다음 로컬 호스트 링크로 자동 접속합니다.

http://localhost:8501

컨트롤 + C로 서버를 종료합니다.
